# Образовательная платформа

Современный веб-сайт образовательной платформы, созданный на основе пользовательского flow-диаграммы.

## Особенности

- 🎨 Современный и отзывчивый дизайн
- 🌙 Поддержка темной темы
- 📱 Мобильная адаптация
- ⚡ Плавные анимации и переходы
- 🔐 Система авторизации
- 📚 Банк заданий
- ✅ Управление заданиями
- 🎁 Система вознаграждений (маркет)
- 👤 Профиль пользователя

## Структура проекта

```
/workspace/
├── index.html      # Основная HTML-структура
├── styles.css      # CSS-стили
├── script.js       # JavaScript-функциональность
└── README.md       # Документация
```

## Запуск

1. Откройте `index.html` в любом современном браузере
2. Или запустите локальный сервер:
   ```bash
   python -m http.server 8000
   # или
   npx serve .
   ```

## Использование

### Навигация
- Используйте верхнее меню для перехода между разделами
- Активный раздел выделен цветом

### Разделы

1. **Авторизация** - вход в систему
2. **Банк заданий** - выбор доступных заданий
3. **Мои задания** - управление выбранными заданиями
4. **Маркет** - система вознаграждений
5. **Профиль** - информация о пользователе

### Интерактивность

- Выбор заданий из банка с подтверждением
- Отказ от заданий с возвратом в банк
- Покупка подарков в маркете
- Модальные окна для подтверждения действий
- Уведомления о статусе операций

### Темная тема
Нажмите на кнопку 🌙 в правом нижнем углу для переключения темы.
