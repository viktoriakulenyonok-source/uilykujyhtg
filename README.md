# Система управления заданиями

Современное веб-приложение для управления задачами с системой аутентификации, банком заданий, профилем пользователя и маркетом подарков.

## 🚀 Особенности

- **Аутентификация**: Безопасная система входа и регистрации
- **Банк заданий**: Поиск и фильтрация доступных заданий
- **Мои задания**: Управление принятыми заданиями с отслеживанием прогресса
- **Профиль**: Личная информация, статистика и настройки
- **Маркет**: Обмен баллов на подарки и награды
- **Детали задания**: Подробная информация о задании с возможностью общения
- **Адаптивный дизайн**: Полная поддержка мобильных устройств

## 🛠 Технологии

- **React 18** - Основной фреймворк
- **React Router** - Маршрутизация
- **Tailwind CSS** - Стилизация
- **Lucide React** - Иконки
- **Vite** - Сборщик проекта

## 📦 Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd task-management-system
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите проект в режиме разработки:
```bash
npm run dev
```

4. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## 🏗 Сборка для продакшена

```bash
npm run build
```

## 🎨 Дизайн

Приложение создано на основе предоставленных CSS стилей с использованием современного дизайна:

- **Цветовая схема**: Синий (#0ea5e9) как основной цвет
- **Типографика**: Inter и SF Pro Text

- **Анимации**: Плавные переходы и hover-эффекты

## 🔧 Настройка

### Переменные окружения

Создайте файл `.env` в корне проекта:

```env
VITE_API_URL=http://localhost:8000
VITE_APP_NAME=TaskManager
```

### Кастомизация

Основные настройки можно изменить в файлах:
- `tailwind.config.js` - цвета, шрифты, размеры
- `src/index.css` - глобальные стили
- `vite.config.js` - настройки сборки

## 📱 Адаптивность

Приложение полностью адаптивно и поддерживает:
- Мобильные устройства (320px+)
- Планшеты (768px+)
- Десктопы (1024px+)
- Большие экраны (1280px+)

## 🚀 Развертывание

### Netlify
1. Подключите репозиторий к Netlify
2. Установите команду сборки: `npm run build`
3. Укажите папку публикации: `dist`

### Vercel
1. Подключите репозиторий к Vercel
2. Автоматическое развертывание при push в main
